<p align="center">
<b>Multiple Machine Learning & NLP Applications built with Python and deployed using Streamlit</b>
<br><br>
This repository showcases real-world ML projects covering

Each project demonstrates the full workflow:
<br>
<b>Data Preprocessing → Model Training → Evaluation → Deployment</b>
</p>

<hr>

<h2>🚀 Live Deployed Applications</h2>

<ul>

<li>
<h3>🧠 Personality Type Predictor</h3>
<p>
Classifies users as <b>Introvert, Extrovert, or Ambivert</b> using a
<b>Logistic Regression</b> model trained on behavioral attributes.
</p>
</li>

<li>
<h3>✉️ Email Spam Predictor</h3>
<p>
Detects spam emails using <b>Natural Language Processing</b> techniques
and a <b>Naive Bayes classifier</b>.
</p>
</li>

<li>
<h3>📝 Review Sentiment Analysis</h3>
<p>
Analyzes product reviews and predicts whether the sentiment is
<b>Positive or Negative</b> using NLP and Machine Learning.
</p>
</li>

<li>
<h3>❤️ Heart Disease Predictor</h3>
<p>
Predicts the likelihood of heart disease using
<b>K-Nearest Neighbors (KNN)</b> based on medical attributes.
</p>
</li>

<li>
<h3>🎓 Academic Performance Predictor</h3>
<p>
Estimates student <b>GPA</b> based on academic behavior and study habits
using a <b>KNN Regression model</b>.
</p>
</li>

<li>
<h3>🏦 Loan Approval Predictor</h3>
<p>
Predicts whether a loan application will be approved using
financial attributes and a <b>Naive Bayes model</b>.
</p>
</li>

<li>
<h3>🩺 Disease Risk Predictor</h3>
<p>
Predicts disease risk using lifestyle indicators such as
<b>BMI, sleep hours, daily steps, blood pressure, and cholesterol</b>
with a <b>Decision Tree classifier</b>.
</p>
</li>

<li>
<h3>💊 Drug Classification Predictor</h3>
<p>
Recommends the most suitable drug for a patient based on
<b>Age, Sex, Blood Pressure, Cholesterol, and Na_to_K ratio</b>
using a <b>Decision Tree model</b>.
</p>
</li>

<li>
<h3>🚗 Old Car Price Predictor</h3>
<p>
Predicts the <b>resale value of used cars</b> using
features such as car name, year, fuel type, transmission,
present price, and kilometers driven using a
<b>Decision Tree Regression model</b>.
</p>
</li>

<li>
<h3>🚚 Food Delivery Time Predictor</h3>
<p>
Predicts the <b>estimated delivery time for food orders</b>
based on features like <b>distance, weather conditions,
traffic level, time of day, vehicle type, preparation time,
and courier experience</b>.
<br>
Built using a <b>Random Forest Regressor</b>.
</p>
</li>

<li>
<h3>📊 Student Exam Score Predictor</h3>
<p>
Predicts the <b>expected exam score of a student</b> using
study-related attributes such as
<b>study hours, class attendance, sleep duration,
sleep quality, study method, and facility rating</b>.
<br>
The model is built using an <b>XGBoost Regressor</b>
and optimized using <b>GridSearchCV</b>.
</p>
</li>
<li>
<h3>🎬 Movie Recommendation System</h3>
<p>
Recommends <b>top 10 similar movies</b> based on user input using 
<b>Natural Language Processing (NLP)</b> techniques.
<br>
Movie descriptions are transformed using <b>TF-IDF vectorization</b> 
and similarity is calculated using <b>Cosine Similarity</b> to find the most relevant movies.
<br>
Built with an interactive <b>Streamlit UI</b> for real-time recommendations.
</p>
</li>

</ul>
